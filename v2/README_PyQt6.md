# PyQt6 version

## Install

```bash
pip install PyQt6
```

## Run

```bash
python pyqt6_page.py
```

UI requirements implemented:
- Left 1/6 area: 7 vertical dots with connecting lines
- Middle-left 1/3 area: split into top and bottom parts
- Right 1/2 area: file content read + display
- Top toolbar left menu: File Tools -> New File / Read File
