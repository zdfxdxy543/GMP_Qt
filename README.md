# GMP_Qt
A visual hardware setup project for GMP based on Python and Qt.

# 20260303 Update
添加了对于ctrl_settings.h和xplt.config.h两个文件的生成向导支持，目前可以提供给f280049c和stm32g431两款芯片使用。  
使用时，可以选择下载[Release](https://github.com/zdfxdxy543/GMP_Qt/releases/tag/GMP_Qt)文件，也可以选择下载源码后运行根目录的`build_with_embed.bat`文件，实现编译。
